git 쓸때 주의사항

작업하기전에 git pull(내려받기) 하고 작업하는걸 추천

올릴때 :
1. git pull받고 (다른사람이 수정한거를 덮어씌워버리는것 방지효과도있음)

2. git add -A  (자기가 수정한거를 전부 원격저장소에 등록)

3. git commit -m '작업내용메시지' (원격저장소에 작업내용 코멘트를 작성하면서 등록)

4. git push (내원격저장소의내용을 사이트 repository로 업로드)

내려받을때

1. git pull (내컴퓨터로 내려받기)

상태확인
1. git status(내 원격저장소와 내작업공간 비교해서 상태확인)
